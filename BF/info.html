Brainf*ck programming language

Yeah, you are right , there is language named like this. Because it has pretty minimal default commands. This f*cks your brain!

Spoiler: this language was not created by me - this is just my javascript implementation

Brainf*ck architecture is quite similar to Turing machine with half-long tape. You basically can move to next byte ">" , move to previous byte "<" and perform other operations with tape and i/o. By default each byte is equal to 0.

There are total of 8 operations (I went over 2 already):

  • "+" increments byte you are pointing at by 1
  • "-" decrements byte you are pointing at by 1
  • "." writes byte you are pointing at as UTF-8 char (for example 48 will write "0")
  • "," reads char in the same fashion way
  • "[" is equal to "while(stack[index]!=0){"
  • "]"is equal to "}"
Very funny, but this is real language and you can perform every finite-state calculations (maybe you would like to hack Enigma :) )
